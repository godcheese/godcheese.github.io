Cheese Api 调试工具 - 一款基于PHP/JavaScript语言开发的轻便API调试工具

发表于 2017-02-08 更新于 2020-09-30 分类于 开发者工具

因为前几次开发app时，发现网络上比如Swagger、apidoc、postman功能都不错，但是收费，可定制性差(待议)，so自己在Google Chrome Store 找到一款叫做Apizza的坑爹api托管网站（现在一调试api就崩溃），word天我那时候写了好多api都在上面，后来无奈自己想到重新开发一款，支持本地部署的api调试工具，采用json可定制配置。几乎所有功能都能通过json配置所得。另还有解析引擎，方便随时随地解析项目文档。 没啥介绍的。已经托管在自己的github/码云主页上，目前项目暂时未完成，暂未私有项目。采用php+js开发。生成工具是php，解析项目引擎是JavaScript。 上几张截图吧。(UI是Google Material Design，暂未用到VueJS、AngularJS，哈哈哈哈) ​
