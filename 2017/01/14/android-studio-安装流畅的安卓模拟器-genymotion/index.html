Android Studio 安装流畅的安卓模拟器 Genymotion

发表于 2017-01-14 更新于 2020-09-30 分类于 技术分享

首先下载安装好Android Studio（http://www.android-studio.org/index.php）。 再前往Genymotion 官网下载，下载需要先登录（https://www.genymotion.com/account/login/），没帐号就先注册（[https://www.genymotion.com/account/create/](https://www.genymotion.com/account/create/)）。 登录后前往下载页面（https://www.genymotion.com/download/）， 以上版本按自己需求下载。 下载好后安装即可。 安装完毕后点击Add下载添加新设备（需登录帐号）即可。 添加设备的Android 系统镜像会自动导入到VirtualBox。 安装好Genymotion后打开Android Studio。 Windows 平台 -> File -> Settings Mac OS X 平台 -> File -> Preferences 选择 Plugins -> Browse Repositories 在搜索框内输入 Genymotion -> Install -> 安装完毕后会重启 Android Studio 即可。 安装完毕后在此处打开Genymotion插件 然后在弹出窗口选择Genymotion安装路径，比如我的E:\Program Files\Genymobile\Genymotion ，设置好后点击OK。 然后重新点击Genymotion插件，打开后会列出当前已下载安装的设备镜像，然后选择，并点击 Start 设备已成功打开。 然后在工栏选择 Run -> Run ‘app’ 将设备选择为 Genymotion 的，然后点击OK。 然后等待 Gradle build 结束，切换会模拟器即可看到已启动当前项目，然后就可以查看控制台调试程序了。
