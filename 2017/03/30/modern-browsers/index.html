推荐的五款现代浏览器 - Google Chrome、Mozilla Firefox、Opera、Apple Safari、Microsoft Edge

发表于 2017-03-30 更新于 2020-09-30 分类于 常用软件 ， 精品资源

Google Chrome（推荐）

软件名称：Google Chrome 软件官网：http://www.google.cn/chrome/browser/ 授权方式：免费 软件语言：简体中文 支持系统：Windows/macOS/Linux/Android

谷歌浏览器，是一个由Google（谷歌）公司开发的开放原始码网页浏览器。该浏览器是基于其他开放原始码软件所撰写，包括WebKit和Mozilla，目标是提升稳定性、速度和安全性，并创造出简单且有效率的使用者界面。

下载地址

官方下载地址：http://www.google.cn/chrome/browser/（下载安装过程建议科学上网） 网盘下载：链接: http://pan.baidu.com/s/1i4CVVGL 密码: gioo

安装说明

网盘内文件说明： ChromeStandaloneSetup64_57.0.2987.133_ 为64位稳定版离线安装包，还没科学上网的电脑建议下载此安装包。 49.0.2623.112_chrome_installer.exe 为Windows XP 系统专用 Windows XP 系统建议下载：49.0.2623.112_chrome_installer.exe（ XP 系统专用） Windows XP SP3 系统以上版本建议下载：_ChromeStandaloneSetup64_57.0.2987.133

Mozilla Firefox

软件名称：Mozilla Firefox 软件官网：http://www.firefox.com.cn/ 授权方式：免费 软件语言：简体中文 支持系统：Windows/macOS/Linux/Android

全球超过5亿用户使用的浏览器，汇聚各种全新特性，可定制性超强。

下载地址

官方下载地址：http://www.firefox.com.cn/download/#more

Opera

软件名称：Opera 软件官网：http://www.opera.com/zh-cn/ 授权方式：免费 软件语言：简体中文 支持系统：Windows/macOS/Linux/Android

Opera浏览器是一款适用于各种平台、操作系统和嵌入式网络产品的高品质、多平台产品。采用Opera Turbo功能极大的改善受限连接上的浏览器速度，例如wifi，手机联网等；新引擎 Opera Presto在渲染内容复杂页面时速度更快，提速40%；全新的可视化标签，根据个人需求自定义自己的快速拨号。

下载地址

官方下载地址：http://www.opera.com/zh-cn/

Apple Safari

软件名称：Safari 软件官网：http://www.apple.com.cn/safari/ 授权方式：免费 软件语言：简体中文 支持系统：Windows/macOS/iOS

Safari使用全新的Nitro引擎展现极其快速的性能表现；提供开发人员工具，可以快速查看页面结构或测试编码的试验性片段等等；运用Top Sites功能通过单个页面中显示的 24 个缩略图；全新 Cover Flow 功能让你能够用图形化的方式查看网站。

下载地址

macOS 下载更新：https://support.apple.com/zh-cn/safari/

安装说明

该浏览器的 Windows 最新版本为2014年更新的，所以 Windows 系统用户不建议使用，仅建议 macOS/iOS 系统使用。

Microsoft Edge

软件名称：Microsoft Edge 软件官网：https://www.microsoft.com/zh-cn/windows/microsoft-edge/ 授权方式：免费 软件语言：简体中文 支持系统：Windows

下载地址

Windows 10 下载升级：https://www.microsoft.com/zh-cn/windows/microsoft-edge/

安装说明

该浏览器为 Microsoft Windows 10 系统专有，其他系统无法安装。
